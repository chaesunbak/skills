# For human whom may read this...

## To save your tokens

### You can run individual tests manually with GUI

Install [Playwright VSCode extension](https://playwright.dev/docs/getting-started-vscode)

or run playwright in ui mode by running `npx playwright test --ui` (or `npm run test:e2e:ui` for short.)

### You can generate tests with codegen

run `npx playwright codegen <url>`

https://playwright.dev/docs/codegen#running-codegen
