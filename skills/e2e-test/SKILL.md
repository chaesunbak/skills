# E2E-testing

## Quick Start

Use Playwright.

If you are setting up E2E tests in the repository for the first time read `references/setup-guide.md`.

If the test is not using test fixture, refactor it to use test fixture. Read `references/fixture-guide.md` for information on using test fixtures.
